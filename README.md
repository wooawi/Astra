🌙 Открой тайны Вселенной каждый день
Твое личное пространство гармонии, энергии и самопознания.
Следи за фазами Луны, узнавай, как они влияют на настроение, эмоции и события вокруг тебя. 
Получай ежедневный гороскоп, вдохновляющие подсказки дня и персональную матрицу дня, которая поможет лучше понять свою энергию, возможности и внутренний баланс.

✨ Что внутри:
• Отображение текущей фазы Луны 
• Лунный календарь с выбором даты
• Ежедневные советы и рекомендации дня
• Гороскоп дня для каждого знака зодиака
• Персональная матрица дня на основе даты рождения
• Профиль пользователя с личной информацией
• Удобная навигация между экранами
• Современный интерфейс в тёмной стилистике



Требования к индивидуальному проекту по дисциплине "Разработка мобильных приложений":
1. Использование языка Kotlin
2.Использование компонентов Jetpack Compose
3. Минимум 3 экрана
4. Локальное хранилище данных (Room)
5. Обработка ошибок (приложение не должно вылетать при неправильных действиях)
6. Состояние экрана (корретная работа состояний загрузки, успеха, ошибок)
7. Архитектурный паттерн MVVM
8. Асинхронность 
9. VCS (github, должно быть минимум 7 коммитов)
10. Использование API
11. CRUD операции
